## 唱法标注

> 这首歌是我非常喜欢的中国原创音乐剧《蝶》里面的唱段
> 内容：讲述了梁山伯被蝶人首领扣押，即将处死，祝英台见他最后一面，二人互诉衷肠  
> 情感基调: 梁山伯以为自己即将狗带，表达出自己对祝英台的真心，即使死了我也依然爱你    
> 


### 歌词 + 解析

```js
女:你站在那个地方      // 那个地方是监狱，  明天就要浸猪笼了

背后是一片荒凉         // 监狱当然荒凉了，总不能给你大别墅把

在这无边的夜色里

不是说有了真相就能看到光亮  

这世界并不在乎有什么真相    // 这个地方还不知道后面的他老爹的阴谋，所以真相应该是 她俩是真心相爱吧

这一刻我的心多盼望

// 梁山伯虽然是个满嘴花里胡哨的渣男
```
